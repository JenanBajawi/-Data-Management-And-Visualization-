# 🚗 Traffic Accident Data Analysis – Cleaning & Visualization

A simple data analysis project that demonstrates how to clean, explore, and visualize traffic accident data.
This project focuses on understanding data patterns and extracting meaningful insights before applying any machine learning models.

This project is intended for learning purposes and building a strong foundation in **data preprocessing and exploratory data analysis (EDA)**.

---

## ✨ Features

* Clean and preprocess raw data
* Perform Exploratory Data Analysis (EDA)
* Visualize data using different plots
* Identify patterns and correlations
* Extract insights from traffic accident data
* Beginner-friendly and well-structured

---

## 🛠 Technologies Used

* Language: Python
* Libraries: Pandas, NumPy
* Visualization: Matplotlib, Seaborn
* Environment: Jupyter Notebook / Google Colab

---

## 🧹 Data Cleaning

Data cleaning is used to prepare the dataset for accurate analysis.

### Steps Performed

* Removed duplicate rows
* Handled missing values
* Detected and removed outliers using IQR
* Converted categorical variables into numerical format

## 🔍 Exploratory Data Analysis (EDA)

EDA helps in understanding the structure and behavior of the dataset.

### Analysis Performed

* Statistical summaries (mean, std, distribution)
* Feature relationships
* Correlation analysis

## 💡 Key Insights

* High traffic density is strongly associated with accidents
* Highways have higher accident rates
* Most accidents occur between ages **25–40**
* Strong relationship between driver age and experience
